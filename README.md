## 서비스 목표 기능
* 패션 정보 공유
* 지도 API를 활용한 주변 옷가게 추천
* 채팅을 통해 트렌드 공유
* 코디 추천 기능


## 주요 기능
* 맵 API를 활용한 추천 기능
* 파이어베이스를 활용한 채팅 기능
* 비트맵을 통한 이미지 업로드
